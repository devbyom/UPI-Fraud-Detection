\## 📌 Project Overview



UPI Fraud Detection System is an AI-powered platform developed to identify fraudulent UPI transactions in real time using Machine Learning algorithms.



The system analyzes transaction details, calculates risk scores, detects suspicious behavior, and provides instant fraud predictions through an interactive cybersecurity-inspired dashboard.

\# ⚙️ Installation



\## Clone Repository



```bash

git clone https://github.com/yourusername/upi-fraud-detection.git

```



\## Move to Project Folder



```bash

cd upi-fraud-detection

```



\## Create Virtual Environment



```bash

python -m venv venv

```



\## Activate Environment



\### Windows



```bash

venv\\Scripts\\activate

```



\### Linux/Mac



```bash

source venv/bin/activate

```



\## Install Dependencies



```bash

pip install -r requirements.txt

```



\---



\# ▶️ Run Project



\## Start Backend



```bash

python main.py

```



\## Start Frontend



```bash

cd frontend

python server.py

```



\## Open Browser



```text

http://localhost:3000

```
